== Fonctionnalités disponibles - SPRINT 4

.Application
--
- En tant qu'utilisateur, je veux pouvoir générer une page HTML de projets tutorés à partir d'un fichier de configuration CSV
--

== Comment lancer l'application MakeOPTIWeb
.Double-cliquer sur l'icône MakeOPTIWeb.jar
.Le fichier HTML généré se trouvera dans le même dossier que le fichier .jar
NOTE: Pour les informations relatives à la sauvegarde de la page HTML générée, se référer à link:docTechniqueProf/#_comment_sauvegarder_l_8217_application_optiweb[la documentation de l'application OptiWeb]
