3. Classes details

3.1. data package

This package contains all the model classes used to represent data. These classes got the same attributes :

  • They have various fields (e.g : class Student contains group, last name and first name)

  • They have matching getters and setters that can read or update these fields

Table 2. Class list
Name Description

Student.java

Structure representing a student

Project.java

Structure representing a project

Subject.java

Structure representing a subject

Teacher.java

Structure representing a teacher

3.2. gui package

This package contains all the classes of the OPTI user interface.

Table 3. Class list

Central class used to listen and manage all user input (e.g. pressed buttons). This is done by getting to know what event occured and from what window/dialog it did.

This class sets up the main window (a JFrame).

This package contains controllers and the core part of the project, the CSV parser.
